# Career Hub IT TOP

Career Hub IT TOP — веб-платформа для взаимодействия студентов, работодателей и администратора в карьерной среде IT TOP.

Проект позволяет студентам создавать резюме, просматривать вакансии, откликаться на них и общаться с работодателями. Работодатели могут регистрироваться, подтверждать email, заполнять профиль компании, публиковать вакансии, просматривать публичные резюме студентов, приглашать кандидатов и вести переписку. Администратор управляет пользователями, студентами, работодателями, вакансиями, резюме и откликами.

Дополнительно в проекте реализован раздел **“Приколы разработчика”** с мини-игрой **Pixel Battle**.

## Возможности проекта

### Студент

- Вход через журнал IT TOP
- Автоматическое создание профиля студента
- Просмотр данных из журнала
- Создание и редактирование резюме
- Публикация, архивирование и удаление резюме
- Скачивание резюме в PDF
- Просмотр вакансий
- Просмотр публичного профиля компании
- Отклик на вакансию с выбранным резюме
- Просмотр истории откликов
- Работа с приглашениями от работодателей
- Чаты с работодателями
- Раздел “Приколы разработчика”
- Участие в Pixel Battle

### Работодатель

- Регистрация через email
- Подтверждение email-кодом
- Вход по email и паролю
- Восстановление пароля через email-код
- Заполнение профиля компании
- Загрузка аватарки компании
- Создание вакансий
- Редактирование вакансий
- Архивирование и восстановление вакансий
- Мягкое удаление вакансий
- Просмотр публичных резюме студентов
- Скачивание PDF-резюме
- Приглашение студента на вакансию
- Просмотр откликов
- Изменение статусов откликов
- Чаты со студентами

### Администратор

- Вход в админ-панель
- Просмотр статистики
- Управление пользователями
- Бан и разбан пользователей
- Подтверждение и отклонение студентов
- Подтверждение и отклонение работодателей
- Просмотр вакансий
- Архивирование, восстановление и удаление вакансий
- Просмотр резюме
- Архивирование, восстановление и удаление резюме
- Просмотр откликов

### Pixel Battle

- Выбор клана студентом
- Пиксельное поле
- Палитра цветов
- Установка пикселя раз в 20 секунд
- Перекрашивание пикселей
- Отображение автора пикселя
- Лидерборд кланов
- Просмотр списка кланов и статистики
- WebSocket-обновления для всех игроков
- Основа для будущих сезонов и наград
